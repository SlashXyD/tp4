Pour lancer l'application du bot, il faudra tout d'abbord se rendre sur le fichier main.py et placer le token dans token="votre token"
Ensuite run le fichier main.py et se rendre sur le serveur
Le prefix utilisé pour ce bot est "/"
Afin de voir toute les commande effecter /aide ou /help
On peut voir un fichier logs.txt se créer afin d'afficher les logs
